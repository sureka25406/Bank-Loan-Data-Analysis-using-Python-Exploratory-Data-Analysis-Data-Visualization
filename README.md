# Bank Loan Data Analysis using Python

## Project Overview
This project focuses on analyzing bank loan data to understand loan distribution, application trends, and customer behavior. The objective of this project is to transform raw loan data into meaningful insights using Python data analysis techniques.

## Problem Statement
Financial institutions receive a large number of loan applications. Analyzing this data helps identify patterns such as loan approval trends, loan amounts, and customer loan behavior. This project aims to explore and visualize loan data to extract useful insights.

For a detailed explanation, the **Problem Statement Presentation (PPT)** is included in this repository.

## Tools & Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

## Project Steps
1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis (EDA)
4. Data Visualization
5. Insights Generation

## Key Analysis Performed
- Monthly Loan Application Trends
- Loan Amount Distribution
- Customer Loan Behavior Analysis
- Data Visualization using charts

## Files Included
- `Bank_Loan_Project.ipynb` – Python analysis notebook
- `financial_loan.xlsx` – Dataset used for analysis
- `Problem_Statement.pptx` – Project problem statement presentation

## Conclusion
This project demonstrates how Python can be used to analyze financial loan data and convert raw data into useful insights through data visualization and exploratory analysis.
